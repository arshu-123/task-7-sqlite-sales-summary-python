# Task 7 - Basic Sales Summary Using SQLite and Python

## Project Title

Sales Summary Analysis Using SQLite and Python

## Objective

Use SQL queries inside Python to extract sales information from a SQLite database and visualize results using a bar chart.

## Dataset Used

Walmart Sales Dataset

## Tools Used

* Python
* SQLite (sqlite3)
* Pandas
* Matplotlib
* Jupyter Notebook

## Analysis Performed

### Database Creation

Created a SQLite database named sales_data.db and imported sales data into a sales table.

### SQL Query

Retrieved total quantity sold and total revenue for each product line using SQL aggregation functions.

### Data Visualization

Created a bar chart to visualize revenue generated by different product lines.

## SQL Query Used

SELECT product_line AS product,
SUM(quantity) AS total_qty,
SUM(total) AS revenue
FROM sales
GROUP BY product_line;

## Key Insights

* Revenue was analyzed across product categories.
* Product lines showed different sales performance levels.
* SQL aggregation helped summarize business data efficiently.

## Outcome

Successfully connected Python with SQLite, executed SQL queries, displayed results using Pandas, and visualized revenue using Matplotlib.
